# Note

This is an experimental project!

A x86-64 kernel written with zig, the goal is to implement a kernel that supports a universal tunnel proxy protocol

## develop tools

- zig(0.11 or higher)
- xorriso
- qemu-full
- bochs

build:

```sh
make run
```
